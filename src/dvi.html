Общее введение

    Программа предоставляет возможность:

  • формировать и обучать модели нейронных сетей
    для классификации рукописных букв латинского алфавита;
  • иметь от 2 до 5 скрытых слоев;
  • использовать сигмоидальную функцию активации для каждого скрытого слоя;
  • иметь возможность обучаться на открытом датасете.
    Дополнительно:
  • реализован перцпетрон двумя способами:
    - в матричном виде (все слои представлены в виде матриц весов)
    - в графовом виде (каждый нейрон представляется в виде некоторого объекта узла,
    связанного с другими узлами по ссылкам)

    make targets:

  • all - сборка программы
  • install - установка программы в катлог src/bin
  • unistall - удаляет программу из каталога установки
  • clean - очистка каталога src от файлов сборки программы
  • dvi - вывод паспорта проекта
  • dist - создание архива .tar с файлами проекта
  • tests - собирает и запускает unit-тесты
  • gcov_report - отображение информации о покрытии backend unit-тестами
